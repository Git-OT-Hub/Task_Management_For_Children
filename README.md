# モチサポ

## サービス概要
子どもの「課題やお手伝い」へのモチベーションをサポートしながら、タスク管理するサービスです。

## 背景と目的
子どもが「宿題をしてくれない」、「勉強をしてくれない」、「家の手伝いをしてくれない」といった課題はどの家庭でも少なからず抱えていると考えています。  
また、子ども自身、やるべき課題が何かを把握しきれていない場合もあると考えています。  
そこで、大人が仕事をして、対価としてお金を貰うように、子どもも課題達成の対価として、親からご褒美を貰うという方法でモチベーションをサポートし、
視覚的にやるべき課題を明確にしておくことで、上記の課題を少しでも解決したいと考えました。
